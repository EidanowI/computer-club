# computer-club
Fullstack react mySQL web app.

## Индексы базы данных

В проекте используются индексы для оптимизации производительности базы данных. **Индексы работают автоматически** - MySQL сам решает, когда их использовать. **Не нужно менять код!**

### Применение индексов

Для применения индексов выполните SQL-файл:
```bash
mysql -u ваш_пользователь -p computer_club < database_indexes.sql
```

### Проверка индексов

Для проверки, что индексы созданы и используются, выполните:
```bash
mysql -u ваш_пользователь -p computer_club < database_check_indexes.sql
```

Или используйте команду `EXPLAIN` перед вашими запросами для проверки использования индексов.

### Как работают индексы

- MySQL оптимизатор **автоматически** выбирает, какой индекс использовать
- Индексы ускоряют запросы с `WHERE`, `JOIN`, `ORDER BY` по индексированным полям
- **Никаких изменений в коде не требуется** - индексы работают "под капотом"
